# Section02 代理人形象预览设计

## 目标

在不重做已确认 UI 结构的前提下，为 `Section 02 / Attribute Settings` 新增一个“代理人形象实时预览”模块。用户调整五维分数后，除了看到右侧文字描述，还能立即看到一个更直观的代理人形象变化结果。

## 已确认约束

- 仅做局部新增，不重做 `section02` 左侧雷达图与开始分析按钮。
- 保留现有五维分数输入逻辑，不改主计算链路。
- 不接回 `age / gender / bmi`。
- 行动能力决定基础人物姿态，其他维度通过身体部位着色表达。
- 人物姿态与比例严格沿用参考图：
  - `5 / 4 / 3`：站立老人
  - `2`：助行器老人
  - `1`：轮椅老人
- 五个说明框统一放在右侧，自上而下固定为：
  - 认知能力
  - 感知能力
  - 心理能力
  - 活力
  - 行动能力
- 每个说明框通过白色虚线引线连接到人物对应身体部位。

## 布局方案

`section02` 保持左右两大区：

- 左侧：保留当前雷达图编辑器与 `Start Analysis` 按钮。
- 右侧：合并原中间空白板块与原右侧说明板块，形成一个新的大板块 `agent visual panel`。

新大板块内部再划分为两列：

- 左列：人物展示区，显示单个当前代理人形象。
- 右列：五个垂直说明框。

推荐比例：

- 左列 `56%`
- 右列 `44%`

这样可以保证人物够大，同时说明框仍有足够宽度容纳文字。

## 视觉风格

人物不沿用参考 PNG 的白底黑线材质，而是重绘成适合当前深色 UI 的科技感版本：

- 白色线稿
- 浅白半透明填充
- 轻微冷色辉光
- 身体局部用综合色卡做渐变发光

人物“动作、朝向、比例、辅助器具关系”保持与参考图一致，但整体呈现风格与当前页面统一。

## 五维映射

身体部位固定映射如下：

- `cognitive`：头脑
- `sensory`：眼 / 耳附近
- `psychological`：心脏位置
- `vitality`：躯干中部
- `locomotor`：腿部 / 下肢

每个部位颜色由该维度当前分数决定，不做多维混色。

## 颜色规则

采用当前综合热力图综合色卡拆成 5 档固定色：

- `1分`：红
- `2分`：橙
- `3分`：中间过渡色
- `4分`：青
- `5分`：蓝

每个身体部位使用：

- 中心较亮的发光色
- 向外扩散的柔和光晕
- 不透明核心 + 半透明外圈

这样既能表达等级，又更符合“代理人状态预览”的视觉语言。

## 说明框内容

每个说明框包含三层信息：

- 维度名称
- 当前分数（1-5）
- 当前等级对应的文字说明

说明框顺序固定，不随部位位置变化，以保持界面整齐稳定。

## 引线规则

从人物对应身体部位到右侧说明框绘制白色虚线引线：

- 线条细
- 白色半透明
- 使用短破折
- 不做箭头

目标是建立清晰的视觉对应关系，而不是制造复杂图示。

## 交互规则

用户拖动 section02 左侧雷达图时，右侧大板块实时更新：

- 修改 `locomotor`：
  - 立即切换人物姿态
- 修改任一维度：
  - 对应身体部位颜色立即更新
- 修改任一维度：
  - 对应说明框分数与描述立即更新

不增加确认按钮，不增加新弹窗。

## 技术实现方向

### HTML

- 调整 `section02` 的中右结构。
- 移除原 `settings-card--agent-empty` 的空白用途。
- 将原 `settings-dimension-list--agent` 并入新的大板块。
- 新增人物预览容器、发光部位层、引线层、说明框容器。

### CSS

- 新增 `agent visual panel` 布局样式。
- 新增人物画布区、说明框、虚线引线、发光层样式。
- 新增桌面与移动端断点下的适配规则。

### JavaScript

- 复用现有 `state.agentModal.draft.capacityScores`。
- 复用现有五维描述文字来源。
- 新增一个专门的代理人形象预览渲染函数。
- 将 `locomotor` 姿态选择与五维身体部位染色拆开处理。

## 风险与边界

- 由于人物需要在前端中可控渲染，最终会是“按参考图严格重绘”的版本，而不是直接把参考 PNG 粘贴进页面。
- 这次仅改 `section02` 的展示层，不改热力图计算、行为规则、报告导出等核心功能。
- 若后续用户希望人物预览也出现在 `section04`，可复用本次结构，但本轮不做。

## 成功标准

- 用户在 `section02` 调整五维分数时，能即时看到代理人姿态与身体部位颜色变化。
- 右侧说明框与人物部位关系清晰。
- 页面风格与当前 `section01/03/04` 保持一致。
- 不影响已有五维雷达图交互和后续分析流程。
